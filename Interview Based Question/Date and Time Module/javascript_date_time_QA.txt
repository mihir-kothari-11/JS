1Ô∏è‚É£ What is Date in JavaScript?
- Date is a built-in object that represents a single moment in time (date + time).
- Internally, it stores time as milliseconds since Jan 1, 1970 (UTC), known as Unix Epoch.

2Ô∏è‚É£ How to create a Date object?
let now = new Date();                   // current date & time
let specific = new Date("2025-08-26");  // from string
let ymd = new Date(2025, 7, 26, 14, 30); // year, month(0-based), day, hr, min

3Ô∏è‚É£ Important Date Methods
let d = new Date();
d.getFullYear();   // year
d.getMonth();      // month (0‚Äì11)
d.getDate();       // day of month (1‚Äì31)
d.getDay();        // day of week (0=Sunday)
d.getHours();      // hours (0‚Äì23)
d.getMinutes();    // minutes
d.getSeconds();    // seconds
d.getMilliseconds();// ms
d.getTime();       // timestamp (ms since 1970)

4Ô∏è‚É£ Set Methods

d.setFullYear(2030);
d.setMonth(11);   // December
d.setDate(25);    // 25th
d.setHours(10, 30, 0); // 10:30:00

5Ô∏è‚É£ Formatting Dates
let d = new Date();
d.toDateString();   // Tue Aug 26 2025
d.toTimeString();   // 14:35:05 GMT+0530
d.toUTCString();    // Tue, 26 Aug 2025 09:05:05 GMT
d.toISOString();    // 2025-08-26T09:05:05.000Z

6Ô∏è‚É£ Working with Timestamps
let now = Date.now();  // current timestamp
let d = new Date(now); // convert timestamp back to date

7Ô∏è‚É£ Difference Between Two Dates
let start = new Date("2025-01-01");
let end = new Date("2025-08-26");
let diff = end - start; // in ms
let days = diff / (1000 * 60 * 60 * 24);
console.log(`Difference: ${days} days`);

8Ô∏è‚É£ Format Date as YYYY-MM-DD
let d = new Date();
let yyyy = d.getFullYear();
let mm = String(d.getMonth() + 1).padStart(2, '0');
let dd = String(d.getDate()).padStart(2, '0');
console.log(`${yyyy}-${mm}-${dd}`);

9Ô∏è‚É£ Why UTC in Date?
- UTC is the global standard time.
- getUTC*() methods give UTC, while normal methods give local time.
