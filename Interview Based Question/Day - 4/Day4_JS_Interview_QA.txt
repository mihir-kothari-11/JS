Answer: A string is a sequence of characters used to represent text, enclosed in single quotes (' '), double quotes (" "), or backticks (` `).

2Ô∏è‚É£ Difference between single, double, and backtick strings?
Answer:
- Single & Double quotes: Basic string literals.
- Backticks (template literals): Allow interpolation (`${}`) and multi-line strings.

3Ô∏è‚É£ Example of template literal?
Answer:
let name = "Mihir";
console.log(`Hello, ${name}!`); // Output: Hello, Mihir!

4Ô∏è‚É£ How to find the length of a string?
Answer:
let str = "JavaScript";
console.log(str.length); // 10

5Ô∏è‚É£ How to access a character in a string?
Answer:
let str = "Hello";
console.log(str[0]); // 'H'
console.log(str.charAt(1)); // 'e'

6Ô∏è‚É£ Are strings mutable in JavaScript?
Answer: No, strings are immutable. Any modification creates a new string.

7Ô∏è‚É£ Example of immutability?
Answer:
let str = "Hi";
str[0] = "h"; 
console.log(str); // "Hi" (unchanged)

8Ô∏è‚É£ Common string methods?
Answer:
- toUpperCase(), toLowerCase()
- trim(), trimStart(), trimEnd()
- slice(), substring(), substr()
- includes(), startsWith(), endsWith()
- replace(), replaceAll()
- split(), concat()

9Ô∏è‚É£ Example: difference between slice and substring?
Answer:
let str = "JavaScript";
console.log(str.slice(0,4)); // "Java"
console.log(str.substring(0,4)); // "Java"
// Difference: slice accepts negative indices, substring does not.

1Ô∏è‚É£2Ô∏è‚É£ Difference between == and === with strings?
Answer:
== compares values only, === compares value + type.
Example:
